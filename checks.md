## 一、违禁标点（零容忍，命中必改）

- 冒号「：」→ 逗号
- 破折号「——」「—」→ 逗号、句号、括号或换句
- 双引号（成对的两种）→「」或干脆不加引号
- 半角逗号「,」在中文句子里 → 全角「，」（英文词组、代码、数字里的逗号保留）。其他中文标点同理用全角

改完再扫一遍标点，修改过程中容易重新引入。

**保留，不要动**：用户的故意标记是风格不是错误，包括「。。」（情绪省略）、「，，」（停顿）、「！！」、括号吐槽。

## 二、AI味词（一出现就暴露是AI写的，换成更口语的）

- 说白了 → 其实就是 / 坦率讲
- 这意味着 / 意味着什么 → 所以呢 / 那结果会怎样
- 本质上 → 说到底 / 其实
- 换句话说 → 你想想 / 也就是说
- 不可否认 → 删掉，正面说
- 综上所述 / 总的来说 → 换成具体的回扣句
- 值得注意的是 / 不难发现 → 删掉，直接说
- 首先…其次…最后 → 自然转场词
- 让我们来看看 / 接下来让我们 → 删掉
- 反直觉 / 反常识 → 删掉，直接说这事怪在哪（"我一开始完全没想到"、"跟我以为的正好反过来"）

## 三、结构性AI味

- **加粗小标题 / markdown标题** → 删，用口语转场句衔接（"说到这个""回到xxx这块"）。例外：纯分条目的方法论帖可用文中数字 1 2 3，但不加粗、不做标题
- **bullet 罗列观点超过3条** → 改成散文叙述
- **教科书开头**（"在当今AI…的时代""随着…的发展"）→ 删，从具体的当下场景切入
- **一上来就抛结论** → 太正式，换个钩子开头
- **空泛工具名**（"AI工具""某个模型"）→ 具体名字（Claude Code、Codex、Seedance 2.0…）
- **编造的假设例子**（"比如有一次…"）→ 用真实细节；没有就写"我还没试过，但想想…"
- **套话结尾**（"希望对你有帮助"）→ 自然收住，或真实CTA（参考帖子2的 coffeechat）

## 四、小红书违禁词（命中会限流甚至不过审，帖子根本没人看到）

和上面三类不同：上面是"暴露AI味"，这类是"帖子被平台压住没人看到"。分两层。

### 第一层 广告法极限词（法律明令，稳定，扫得全）

绝对化用语，扫到改相对表达或删程度副词：

- **最字类**：最、最佳、最好、最大、最高、最强、最先进、最便宜、最受欢迎、最新、最高级、最高端
- **一字类**：第一、唯一、独一无二、全国/世界/全球/行业/销量第一、第一品牌、NO.1、TOP1、独家、首个、首选、首发、前所未有、史无前例、绝无仅有
- **级极类**：国家级、世界级、国际级、顶级、顶尖、极致、极品、终极、巅峰
- **权威类**：权威、王牌、领袖、领导品牌、冠军、王者、无敌、遥遥领先
- **绝对类**：绝对、100%、百分百、保证、必、一定、永久、彻底、史上最

改法举例："最好用的工具"→"我用过最顺手的之一"；"100%有效"→"我自己跑下来有效"

### 第二层 平台高危类别（账号定制，需自己维护）

小红书不公开这层词库，网上的"违禁词大全"都不可信。下面是按这个账号内容划的高危区，**被限流后回来把真实命中的词补进对应类别，标日期**，让它越用越准。

- **B 效果/收益承诺**（写流量/SEO效果时高危）：月入过万、躺赚、稳赚、轻松赚、暴富、零风险、保本、包赚、包过 → 改成个人经验陈述。注意「三个月7倍增长」这种真实复盘数据**可以用**，是已发生的事实，不是对读者的承诺
- **C 站外导流**（最敏感，几乎必限流）：微信、vx、➕V、私我、扫码、二维码、公众号、QQ、联系方式 → 站内引导可以（"后台dm我"、评论区留言）
- **D 营销黑话**（部分被打压）：引流、截流、收割、割韭菜、私域、起号、养号、刷量 → 中性说法或英文（distribution、visibility、outreach）
- **E 时政敏感**（写国际新闻/欧盟/中美时高危，但这是用户的内容角度，不删观点）：领导人姓名、敏感地区/冲突的明确指代 → 降识别度而非删判断，用谐音/拆字/英文缩写（参考帖子1的 yi/e/wu）。清楚时政帖流量上限天生低，发前权衡值不值
**通过标准**：第一层 + B/C/D 零命中；E 类命中则确认已模糊化，或已确认接受限流风险。
